# 📅 찢어 쓰는 일력 - 모바일 앱

손으로 달력을 찢는 것처럼 상호작용하는 React Native 모바일 애플리케이션입니다.

## ✨ 주요 기능

### 🗓️ 일일 달력 모드
- **터치 & 스와이프 찢기**: 버튼 터치 또는 아래로 스와이프해서 달력 찢기
- **3D 애니메이션**: 실제 달력을 찢는 듯한 3D 회전 및 이동 애니메이션
- **햅틱 피드백**: 찢을 때 진동으로 촉각 피드백 제공
- **하루 하나씩**: 하루에 날짜 하나만 보여주고, 찢으면 다음 날로 자동 이동
- **계절별 이모지**: 계절에 따라 다른 이모지로 날짜 표시 (🌸봄, ☀️여름, 🍂가을, ❄️겨울)
- **찢어진 날짜 추적**: 이미 찢어진 날짜는 다시 찢을 수 없음

### 📅 월별 달력 모드
- **전체 달력 보기**: 한 달 전체를 한눈에 볼 수 있는 터치 친화적 달력
- **찢어진 날짜 표시**: 찢어진 날짜는 빨간색으로 표시되며 ✂️ 아이콘 추가
- **월 네비게이션**: 터치로 이전/다음 월로 자유롭게 이동 가능
- **날짜 선택**: 달력에서 날짜를 터치하면 해당 날로 바로 이동

## � 모바일 전용 기능

- **터치 최적화**: 모든 인터랙션이 터치에 최적화되어 있음
- **햅틱 피드백**: 찢기 동작 시 진동 피드백
- **스와이프 제스처**: 아래로 스와이프해서 달력 찢기 가능
- **반응형 디자인**: 다양한 화면 크기에 자동 적응
- **네이티브 성능**: React Native로 구현된 부드러운 애니메이션

## �🚀 사용법

1. **일력 찢기**: 
   - "📄 오늘을 찢어내기" 버튼을 터치
   - 또는 달력을 아래로 스와이프
2. **햅틱 피드백**: 찢을 때 진동으로 촉각 피드백 받기
3. **애니메이션 감상**: 달력이 찢어지는 3D 애니메이션을 즐기세요
4. **자동 이동**: 찢기가 완료되면 자동으로 다음 날로 이동
5. **전체 달력**: 우측 상단의 "📅 전체 달력" 버튼으로 월별 보기 전환
6. **날짜 선택**: 월별 보기에서 원하는 날짜를 터치해 해당 날로 이동

## 💾 데이터 저장

- 찢어진 날짜 정보는 기기의 AsyncStorage에 자동 저장
- 앱을 다시 열어도 이전에 찢어진 날짜들이 기억됨

## 🎨 디자인 특징

- **모던한 그라데이션**: 아름다운 보라색 그라데이션 배경
- **터치 친화적 UI**: 모든 버튼과 인터랙션이 터치에 최적화
- **반응형 디자인**: 다양한 스마트폰 화면 크기에서 최적화된 사용자 경험
- **부드러운 애니메이션**: 모든 상호작용에 부드러운 네이티브 애니메이션

## 🔧 기술 스택

- **React Native**: 크로스플랫폼 모바일 앱 개발
- **Expo**: 빠른 개발 및 배포를 위한 플랫폼
- **React Native Reanimated**: 고성능 애니메이션
- **React Native Gesture Handler**: 터치 제스처 처리
- **AsyncStorage**: 로컬 데이터 저장
- **Expo Haptics**: 햅틱 피드백

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# Expo 개발 서버 시작
npm start

# iOS에서 실행 (macOS 필요)
npm run ios

# Android에서 실행
npm run android

# 웹에서 실행 (테스트용)
npm run web
```

## 📱 앱 빌드

```bash
# APK 빌드 (Android)
expo build:android

# IPA 빌드 (iOS, macOS 필요)
expo build:ios
```
