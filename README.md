# 📅 찢어 쓰는 일력 어플

손으로 달력을 찢는 것처럼 상호작용하는 인터랙티브한 일력 애플리케이션입니다.

## ✨ 주요 기능

### 🗓️ 일일 달력 모드
- **찢기 애니메이션**: 실제 달력을 찢는 듯한 3D 애니메이션 효과
- **하루 하나씩**: 하루에 날짜 하나만 보여주고, 찢으면 다음 날로 자동 이동
- **계절별 이모지**: 계절에 따라 다른 이모지로 날짜 표시 (🌸봄, ☀️여름, 🍂가을, ❄️겨울)
- **찢어진 날짜 추적**: 이미 찢어진 날짜는 다시 찢을 수 없음

### 📅 월별 달력 모드
- **전체 달력 보기**: 한 달 전체를 한눈에 볼 수 있는 달력
- **찢어진 날짜 표시**: 찢어진 날짜는 빨간색으로 표시되며 ✂️ 아이콘 추가
- **월 네비게이션**: 이전/다음 월로 자유롭게 이동 가능
- **날짜 선택**: 달력에서 날짜를 클릭하면 해당 날로 바로 이동

## 🚀 사용법

1. **일력 찢기**: 메인 화면에서 "📄 오늘을 찢어내기" 버튼을 클릭
2. **애니메이션 감상**: 달력이 찢어지는 3D 애니메이션을 즐기세요
3. **자동 이동**: 찢기가 완료되면 자동으로 다음 날로 이동
4. **전체 달력**: 우측 상단의 "📅 전체 달력" 버튼으로 월별 보기 전환
5. **날짜 선택**: 월별 보기에서 원하는 날짜를 클릭해 해당 날로 이동

## 💾 데이터 저장

- 찢어진 날짜 정보는 브라우저의 로컬 스토리지에 자동 저장
- 앱을 다시 열어도 이전에 찢어진 날짜들이 기억됨

## 🎨 디자인 특징

- **모던한 그라데이션**: 아름다운 보라색 그라데이션 배경
- **글래스모피즘**: 반투명 유리 효과를 활용한 세련된 UI
- **반응형 디자인**: 모바일과 데스크톱 모두에서 최적화된 사용자 경험
- **부드러운 애니메이션**: 모든 상호작용에 부드러운 전환 효과

## 🔧 기술 스택

- **React 18**: 최신 React 기능 활용
- **CSS3**: 3D Transform, Keyframe Animation, Grid Layout
- **Local Storage**: 데이터 영속성
- **Responsive Design**: 모바일 친화적 설계
